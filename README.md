# GameTech Platform

`gametech_platform` is a standalone minimal project scaffold extracted from the current CRMEB workspace for a game resource and technical service platform.

## Structure

- `server/`: Spring Boot backend skeleton
- `admin/`: Vue admin dashboard skeleton
- `app/`: uni-app client skeleton
- `docs/`: product and architecture notes

## MVP Scope

- user login and role extension points
- banner and homepage basic data
- engineer onboarding model placeholders
- resource center placeholder
- service order placeholder
