# Architecture

## Positioning

This project is not a mall. It is a platform for:

- paid resources
- technical services
- engineer onboarding and dispatch

## Recommended Module Order

1. system
2. user
3. banner
4. engineer
5. resource
6. service-order
7. finance

## What Was Intentionally Removed

- coupon
- seckill
- bargain
- combination
- distribution
- logistics
- physical product workflow

## Shared Foundation Retained Conceptually

- authentication
- unified api response
- redis cache
- mysql persistence
- upload abstraction
- admin/app separation
